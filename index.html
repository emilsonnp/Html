Lista de Tags do HTML

  • <!DOCTYPE> - Define o tipo de documento
  • <html> - Elemento raiz de um documento HTML
  • <head> - Contém metadados sobre o documento
  • <title> - Define o título do documento
  • <body> - Contém o conteúdo visível do documento

Esta é uma lista de algumas das tags mais comuns do HTML.

Outras Tags Importantes

  • <ul> - Define uma lista não ordenada
  • <ol> - Define uma lista ordenada
  • <a> - Define um hyperlink
    • Final da pagina
  • <li> - Define um item de lista
  • <p> - Define um parágrafo
  • <font> - Define uma fonte
    • Tag Font foi depreciada no HTML5, mas ainda pode ser usada para definir a aparência do texto.
  • <del> - Define texto deletado
  • <abbr> - Define uma abreviação

Tag Mark

  • <mark> - Define texto destacado

Tag Small e Tags I e U

  • <small> - Define texto em um tamanho menor
  • <i> - Define texto em itálico
  • <u> - Define texto sublinhado
Tag Strong e Tags A e HR
  • <strong> - Define texto em negrito
  • <a> - Define um hyperlink - HTML DEVELOPER
  • <hr> - Define uma linha horizontal
Tags Sup e Sub Tag Blockquote
  • <sub> - Define texto sobrescrito
  • <sup> - Define texto subescrito
  • <blockquote> - Define uma citação em bloco
